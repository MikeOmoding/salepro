# Salepro

Salepro is a management system made in laravel to manage a vendors shop by tracking sales and stock. 

## Features
Salepro manages the following.
- Stock.
- Purchases.
- Sales.
- Expenses.
- Returns.
- Accounting.
- Human resource.
- Reports.
